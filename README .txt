-> Planilha Vintage

- O projeto planilha vintage foi feito no Visual Studio 2013
- O projeto utiliza a biblioteca Irvine
- O projeto possui 5 arquivos . asm, cada um possui uma explificação 
no ínicio, indicando para que foi criado.
- Sendo que o arquivo principal, é o PlanilhaVintage.asm
- Todos os procedimentos utilizados estão devidamente comentádos


----------------USANDO A PLANILHA-------------------------------

-> Ao iniciar o executável, a planilha se abrirá automaticamente.

-> Para utilizar a planilha basta seguir as instruçoes presentes
na vizualização da planilha

-> Quando for digitar uma fórmula, as colunas devem estar em letra
maiúscula e a formula deve ser desse tipo "=A2*A3"

-> Quando for digitar uma função realizar como no exemplo abaixo: 
	MODO -> M=A2:A4
	MENOR -> N=B2:B5
	CONT.VALORES -> C=C4:C8
